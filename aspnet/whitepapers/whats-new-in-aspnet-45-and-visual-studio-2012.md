Form* o *HttpRequest.Files*.</span><span class="sxs-lookup"><span data-stu-id="812a3-190">For example, if some upstream code in the pipeline has already read the request entity using *GetBufferedInputStream*, you can still use *HttpRequest.Form* or *HttpRequest.Files*.</span></span> <span data-ttu-id="812a3-191">Ciò consente di eseguire l'elaborazione asincrona in una richiesta (ad esempio, il caricamento di un file di grandi dimensioni a un database di streaming), ma le pagine aspx ancora esecuzione e controller MVC ASP.NET in un secondo momento.</span><span class="sxs-lookup"><span data-stu-id="812a3-191">This lets you perform asynchronous processing on a request (for example, streaming a large file upload to a database), but still run .aspx pages and MVC ASP.NET controllers afterward.</span></span>

<a id="_Toc318097375"></a>
